== Description ==

A simple WordPress Block plugin to shout-out your love for Brave Browser. The only functionality of this plugin is for webmasters to insert a Block in WordPress Widget that'll create an awareness about [Brave Browser](https://brave.com/) and the importance of **Privacy** and **Tracking Free Internet Experience**.

= For Gutenberg Editor: Using Blocks =

- For all Gutenberg Block Editor users, you can simply add the functionality by simply inserting the block, anywhere in the widgets page as per your website design.

Users who doesn't use the Gutenberg Block editor can still make use of this plugin along with the Classic Editor or Classic Widget.

= For Classic Editor: Shortcode Application =

- You can use the shortcode `[with_love_brave]` anywhere in the widget section to insert Love Brave Browser shout-out.

== Installation ==

This section describes how to install the plugin and get it working.

- Install the plugin through the WordPress plugins screen directly.
- Activate the plugin through the ‘Plugins’ screen in WordPress



== Frequently Asked Questions ==

= Will it work with any theme =
Yes the plugin should work with any theme.

= I've disabled Gutenberg Block Widget. Will it still work? =
Yes. This plugin does support shortcode feature. So even if you don't have the block widget you can still use it with shortcode function. Use the shortcode `[with_love_brave]`.
